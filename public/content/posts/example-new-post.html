This is an example blog post to show you how to create new posts for this blog.

Basic Structure

Each post needs:

  • A unique slug (filename)
  • Metadata in JSON format
  • Content wrapped in <div class="html-content">

Formatting Options

You can use bold text, italic text, and links.

Subheadings

And different levels of headings for organization.

You can also include blockquotes for important quotes or highlights.

Publishing

To publish a post:

  1. Set "is_published": true
  2. Set a publish_date
  3. Commit and push to GitHub

That's it! Your new post will appear on the blog.
